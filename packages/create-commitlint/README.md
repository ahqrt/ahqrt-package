# create-commitlint

## Description

这是在项目里面生成commitlint配置的命令行工具，在项目的根目录下面执行以下命令即可.

## Usage

```sh
$ npx @ahqrt/create-commitlint
$ yarn create @ahqrt/create-commitlint
$ npm init @ahqrt/create-commitlint
```

## Install global

```sh
# install
$ npm install @ahqrt/create-commitlint -g
# usage
$ create-commitlint
```
